## Como debe usarlos el asistente

Ver `knowledge-routing-rules.md` para las reglas completas. En resumen:

1. **Pregunta específica de carrera →** `oferta-academica.md` o `sources/Base_Actualizada_Universidad_Latino.csv`
2. **Pregunta de proceso/política →** `faq.md`, `contacto-campus.md`
3. **Mensaje vago/objeción →** `faq-conversacional.md`
4. **Sin respuesta en ninguna capa →** NO inventar. Escalar a asesor.
