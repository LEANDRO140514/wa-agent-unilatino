# Admisiones — Universidad Latino

**Fuente:** source-of-truth validado (`proceso_admision.csv`, `documentos_inscripcion.csv`, `contacto_institucional.csv`).

## Proceso de admisión

1. **Orientación** — Orientación sobre la carrera de interés.
2. **Revisión documental** — Revisión de requisitos y documentación.
3. **Formato de inscripción** — Llenado del formato de inscripción.
4. **Elección de pago** — Elección del tipo de pago (anual o por periodo).
5. **Pago y entrega** — Realización del pago y entrega de documentos.

**No se requiere examen de admisión** (validado en políticas institucionales).

## Documentos para inscripción

1. Acta de nacimiento
2. Certificado de bachillerato (original y copia) — se puede iniciar con constancia si el certificado está en trámite; entregar antes de iniciar clases en septiembre
3. CURP
4. Comprobante de domicilio (copia)

## Formas de pago documentadas

- Pago en mensualidades (disponible)
- Pago anual con descuento — `PENDIENTE_VALIDACION_ADMISIONES` (monto)
- Pago semestral con descuento — `PENDIENTE_VALIDACION_ADMISIONES` (monto)
