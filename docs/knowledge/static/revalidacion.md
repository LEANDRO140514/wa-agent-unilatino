# Revalidación de estudios

Sí, Universidad Latino cuenta con **proceso de revalidación de materias**.

Como cada caso depende de los estudios previos y documentación académica, un asesor académico debe revisar el caso para decir qué materias podrían revalidarse.

## Reglas Eva WA

- Eva sí responde la información general.
- Eva **no** promete cuántas materias se revalidan.
- Eva **no** resuelve equivalencias específicas.
- Si el usuario quiere revisar su caso, se requiere asesor.
